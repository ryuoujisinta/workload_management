# 工数管理アプリ

- 管理者画面
    - ユーザー登録画面
    - タスク登録画面
        - 月ごとに登録
    - タスク承認画面
        - 承認申請された工数を承認する
        - 承認申請された工数を却下する
- ユーザー画面
    - ログイン画面
        - ログイン
        - パスワードを忘れた場合
    - パスワード変更画面
    - タスク登録画面
        - 管理者が登録したタスクのうち、必要なものを登録
    - 工数登録画面
        - タスク登録画面で登録したタスクを表示
        - 1週間ごとの画面
        - 前週・次週の画面に移動できる
        - 以下のボタンは日ごとに表示
            - 登録
            - 一時保存
            - 登録取り消し
